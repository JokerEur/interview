Задание 5.

перк SHIELDING уменьшает входящий урон по игроку в 2 раза.

На мой взгляд даннну задачу не возможно решить, так как отсутствует функция , которая бы возвращала заначение урона полученое игроком.

ТЗ для программиста :
необходимо реализовать функцию, которая будет возвращать целочисленное заначение урона полученое игроком или противком.
функция должна принимать в себя стоку(std::stirng TargetName) как параметр и возвращать int Damage.

функция должна выглядить как-то так:

int GetDamage(std::stirng TargetName){
    if( TargetName == "Me"){
        return Player.Damage;
    }else if( TargetName == "Enemy"){
        return Enemy.Damage;
    }
    throw std::invalid_argument("GetDamage arguments must be Me or Enemy");
}
